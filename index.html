How to define a table?

<tr>
Used to define a row in a table

<th>
Used to define a header cell; typically bolded and centered

<td>
Used to define a standard data cell within a row

<thead>
Used for group header content in a table

<tfoot>
Used for groups footer content in a table

<caption>
Used to add a title or caption to a table

Difference Between colspan and rowspan
Colspan is used to span a cell across multiple columnns while rowspan is used to span a cell over multiple rows

Difference between cells spacing and padding
Cells spacing is the attribute used to control space between individual cells while cells padding is the attribute that controls the space between a cell's content and the cell's border.

How to define table borders using CSS
Can use the border property on the <table>, <th>, and <td> elements. CSS will apply the characteristics listed, such as whether the border is a solid line, the color, and width to the table, table header cells, and table data cells.

How to control broder widths with CSS
Specify any desired width in border property; will apply to listed elements such as <table>, <th>, and <td>, can be seen in the table, table headers, and table data cells
